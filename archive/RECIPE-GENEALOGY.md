# Regional Recipe Genealogy

> *Tracing the family tree of 番茄鸡蛋面 across time and place*

## 食谱谱系 | The Recipe Family Tree

---

## Overview

Like a family genealogy, recipes have ancestors, descendants, and regional branches. This document traces the evolution of Chinese tomato noodle soup from its origins to its modern regional variations.

---

## The Proto-Recipe (Ur-Recipe)

### What Came Before

Before 番茄鸡蛋面 existed, its parent dishes were:

```
              TOMATO ADOPTION (1920s-40s)
                        │
         ┌──────────────┼──────────────┐
         │              │              │
    番茄炒蛋        番茄蛋汤        传统面条
    Stir-fried     Tomato egg      Traditional
    tomato egg     soup            noodles
         │              │              │
         └──────────────┼──────────────┘
                        │
                        ▼
              番茄鸡蛋面 (c. 1940s-50s)
              Tomato Egg Noodles
```

### Parent Dishes

#### 番茄炒蛋 (Stir-fried Tomato Egg)
- **First documented:** ~1930s Shanghai
- **Characteristics:** Tomatoes and scrambled eggs stir-fried together
- **Contribution to noodle dish:** The flavor combination, cooking technique

#### 番茄蛋汤 (Tomato Egg Drop Soup)
- **First documented:** ~1940s
- **Characteristics:** Light tomato broth with egg ribbons
- **Contribution to noodle dish:** The soup format, egg drop technique

#### 传统面条汤 (Traditional Noodle Soup)
- **Characteristics:** Noodles in various broths
- **Contribution to noodle dish:** The noodle component, soup-noodle format

## The First Generation (1940s-1960s)

### The "Standard" Version Emerges

| Aspect | Early Version |
|--------|---------------|
| **Tomatoes** | Fresh, peeled, stir-fried |
| **Eggs** | Scrambled, mixed in |
| **Noodles** | Whatever available |
| **Broth** | Water + tomato liquid |
| **Seasoning** | Salt, sometimes soy sauce |

### Where It Crystallized

| Location | Context |
|----------|---------|
| Shanghai | Home cooking, Western influence |
| Beijing | Northern adaptation |
| Work unit cafeterias | Mass production standardization |

### Why This Version Dominated

1. **Simplicity** - Few ingredients, easy technique
2. **Speed** - Quick to prepare
3. **Economy** - Cheap ingredients
4. **Nutrition** - Protein + vitamin C
5. **Adaptability** - Works with available ingredients

## Northern Branch (北方系)

### Beijing Style (北京风格)

**Emergence:** ~1970s
**Distinctive features:**

| Element | Beijing Style |
|---------|---------------|
| **Noodle** | Hand-cut (刀切面) or knife-shaved |
| **Tomato** | Cooked to thick gravy (打卤) |
| **Egg** | Egg drop ribbons (蛋花) |
| **Broth** | Thick, poured over noodles |
| **Extra** | May include minced meat |

**Key technique:** 打卤 (dǎlǔ) - making a thick gravy/topping

**Ancestry:**
```
Traditional 打卤面 (gravy noodles)
         +
番茄炒蛋 technique
         =
番茄鸡蛋打卤面
```

### Shandong Style (山东风格)

**Emergence:** ~1980s
**Distinctive features:**

| Element | Shandong Style |
|---------|----------------|
| **Noodle** | Hand-pulled (拉面) |
| **Tomato** | Chunky, less cooked |
| **Egg** | Scrambled large curds |
| **Broth** | Clear to medium |
| **Extra** | Garlic, scallion prominent |

**Ancestry:**
```
Shandong 拉面 tradition
         +
Standard tomato-egg
         =
番茄鸡蛋拉面
```

### Northeastern Style (东北风格)

**Emergence:** ~1980s
**Distinctive features:**

| Element | Northeastern Style |
|---------|-------------------|
| **Noodle** | Thick, chewy (东北大面) |
| **Tomato** | Long-cooked, sweet |
| **Egg** | Scrambled, hearty portions |
| **Broth** | Rich, often with pork stock |
| **Extra** | Larger portions overall |

## Sichuan-Chongqing Branch (川渝系)

### Chengdu Style (成都风格) - 煎蛋面

**Emergence:** ~1980s, Huaxing Street
**Distinctive features:**

| Element | Chengdu Style |
|---------|---------------|
| **Noodle** | Fresh alkaline (碱水面) |
| **Tomato** | Cooked to thick paste (出沙) |
| **Egg** | Whole fried egg (煎蛋), runny yolk |
| **Broth** | Thick tomato soup |
| **Extra** | Optional chili oil, pickled vegetables |

**Key innovation:** Using 煎蛋 (fried egg) instead of scrambled

**Ancestry:**
```
Chengdu late-night noodle culture
         +
Standard tomato base
         +
煎蛋 (fried egg) technique
         =
华兴街番茄煎蛋面
```

### Chongqing Style (重庆风格)

**Emergence:** ~1990s
**Distinctive features:**

| Element | Chongqing Style |
|---------|-----------------|
| **Noodle** | Small noodles (小面) base |
| **Tomato** | Thick, slightly spicy |
| **Egg** | Fried or scrambled |
| **Broth** | Can include 红油 (chili oil) |
| **Extra** | Sichuan peppercorn optional |

**Ancestry:**
```
重庆小面 (Chongqing small noodles)
         +
Tomato-egg combination
         =
番茄鸡蛋小面
```

## Shanghai Branch (沪系)

### Shanghai Style (上海风格)

**Emergence:** ~1940s-50s (among earliest)
**Distinctive features:**

| Element | Shanghai Style |
|---------|----------------|
| **Noodle** | Fine noodles (龙须面, 挂面) |
| **Tomato** | Lighter cooking, some chunks visible |
| **Egg** | Delicate ribbons |
| **Broth** | Light, clear, slightly sweet |
| **Extra** | May add sugar for balance |

**Key characteristic:** Delicacy and balance over heartiness

**Ancestry:**
```
Shanghai Western-influenced cooking
         +
阳春面 (plain noodle soup) tradition
         +
Sweet-savory Shanghai palate
         =
上海番茄鸡蛋面
```

---

## Canton-Hong Kong Branch (粤港系)

### Hong Kong Style (港式) - 茶餐廳版

**Emergence:** ~1970s
**Distinctive features:**

| Element | Hong Kong Style |
|---------|-----------------|
| **"Noodle"** | Often macaroni (通粉) |
| **Tomato** | Stewed, sweet-tangy |
| **Egg** | Scrambled or fried |
| **Broth** | Soup or dry-style |
| **Extra** | May include spam, ham |

**Key innovation:** Using Western macaroni instead of Chinese noodles

**Ancestry:**
```
Cha chaan teng (茶餐廳) fusion culture
         +
British colonial influence
         +
Tomato-egg concept
         =
港式番茄通粉/麵
```

### Cantonese Style (广东风格)

**Emergence:** ~1980s
**Distinctive features:**

| Element | Cantonese Style |
|---------|-----------------|
| **Noodle** | Egg noodles (蛋面) |
| **Tomato** | Lighter touch |
| **Egg** | Smooth, delicate |
| **Broth** | Clear, relies on good stock |
| **Extra** | Emphasis on freshness |

## Northwest Branch (西北系)

### Xi'an Style (西安风格)

**Emergence:** ~1990s
**Distinctive features:**

| Element | Xi'an Style |
|---------|-------------|
| **Noodle** | Wide, hand-pulled (扯面, biangbiang面) |
| **Tomato** | Standard preparation |
| **Egg** | Scrambled or fried |
| **Broth** | Can be soup or dry |
| **Extra** | Oil-splashed (油泼) option |

**Ancestry:**
```
Shaanxi 油泼面 tradition
         +
Tomato-egg combination
         =
番茄鸡蛋扯面/油泼面
```

### Xinjiang Style (新疆风格)

**Emergence:** ~1990s
**Distinctive features:**

| Element | Xinjiang Style |
|---------|----------------|
| **Noodle** | Laghman (拉条子) - pulled, thick |
| **Tomato** | Heavy use, stewed |
| **Egg** | Can be included |
| **Protein** | Often lamb (羊肉) |
| **Extra** | Cumin, peppers |

**Ancestry:**
```
Uyghur laghman tradition
         +
Tomato (already common in Xinjiang)
         +
Chinese tomato-egg concept
         =
新疆番茄拉条子
```

### Lanzhou Style (兰州风格)

**Note:** Traditional Lanzhou beef noodles (兰州牛肉面) do NOT include tomatoes. However, modern variations sometimes add tomato as an option.

**Modern adaptation:**
| Element | Lanzhou Variation |
|---------|-------------------|
| **Noodle** | Hand-pulled (拉面) |
| **Tomato** | Optional addition |
| **Egg** | Optional |
| **Broth** | Modified from clear beef broth |

**Status:** Not traditional; modern fusion

## Taiwan Branch (台湾系)

### Taiwan Beef Noodle Soup Connection

**Emergence:** ~1950s-60s
**Context:** Mainland soldiers brought to Taiwan created beef noodle soup

| Element | Taiwan Style |
|---------|--------------|
| **Noodle** | Thick wheat noodles |
| **Tomato** | Used in 红烧 (red-braised) base |
| **Protein** | Beef chunks |
| **Broth** | Rich, red-braised |
| **Extra** | Soy sauce, spices |

**Key point:** Taiwan's famous beef noodle soup uses tomato as a base ingredient, connecting to the broader tomato-noodle tradition.

**Ancestry:**
```
Mainland 红烧 technique
         +
Taiwan local ingredients
         +
Military veteran cooking
         =
台湾红烧牛肉面 (with tomato base)
```

### Taiwan Home-Style

| Element | Taiwan Home Style |
|---------|-------------------|
| **Noodle** | Various |
| **Tomato** | Standard preparation |
| **Egg** | Scrambled or fried |
| **Broth** | Soup style |
| **Extra** | Often sweeter |

## Identifying Recipe Ancestry

### Questions to Determine Origin

When you encounter a tomato egg noodle recipe, ask:

1. **Egg style?**
   - Scrambled → Standard, Northern, Shanghai
   - Fried whole → Chengdu
   - Egg drop → Northern, Shanghai

2. **Tomato preparation?**
   - Thick paste → Northern 打卤, Chengdu
   - Chunky → Shandong, some Shanghai
   - Heavily stewed → Xinjiang, Hong Kong

3. **Noodle type?**
   - Hand-pulled → Shandong, Lanzhou, Xinjiang
   - Wide, thick → Xi'an, Northeastern
   - Fine, thin → Shanghai
   - Macaroni → Hong Kong
   - Alkaline → Chengdu

4. **Broth style?**
   - Thick gravy → Northern 打卤
   - Light soup → Shanghai
   - Minimal (dry) → Xi'an (油泼)

5. **Additional ingredients?**
   - Chili oil → Sichuan/Chongqing
   - Spam/ham → Hong Kong
   - Lamb → Xinjiang
   - Cumin → Xinjiang, some Xi'an
